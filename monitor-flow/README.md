# @abstractuic/monitor-flow

Shared React components for **monitoring flows and agent cycles** across AbstractFramework web clients.

This package is currently developed in-repo and consumed by:
- `abstractflow/web/frontend`
- `abstractobserver`

## Components
- `AgentCyclesPanel`: groups effect steps into “cycles” (LLM call → tool calls → observe) and renders a compact, inspectable view.
